Modern Web Developement with Angular

By Patrick Fritzl

Kurzzusammenfassung

Ein Kurs zum Einstieg in das moderne Entwickeln mit Angular, HTML, CSS und JavaScript/Typescript

Mobile First

Die Website wird in Hinsicht zu Mobilen Geräten erstellt, sowie auch für Tablets und Desktop

Clean UI

Mittels PrimeNG in Kombination mit selbstgeschriebenem CSS und HTML werden moderne User-Interface Elemente erstellt.

Interactivity

Durch Databinding und Typescript werden ganz leicht Variablen eingelesen bzw. ausgegeben im Browser

Zeit

Mindestens 4 Einheiten zu je 2h. Je nach Nachfrage mehr möglich

Benötigte Kenntnisse

Grundlegende Programmierkenntnisse (egal welche Sprache)

Grundlegendes HTML & CSS Verständnis
